HEADLINE : Bestuur compleet

De Amerikaan Jim Easton is in Salt Lake City gekozen tot vice-president van het Internationaal Olympisch Comite ( IOC ) . Hij kreeg de voorkeur ( 52-32 stemmen ) boven de Canadees Paul Henderson. Er was een post vacant door het vertrek van Keba Mbaye uit Senegal. Met de verkiezing van Easton is het 15 personen tellende dagelijks bestuur van het IOC weer compleet. Het bestuur bestaat uit : voorzitter : Rogge ( Bel ) , vice-president : Easton ( VS ) , Gosper ( Aus ) , Bach ( Dui ) , Smirnov ( Rus ) , leden : Zhenlian He ( Chn ) , Lindberg ( Zwe ) , Carraro ( Ita ) , Nikolaou ( Gri ) , Khoury ( Liba ) , Oswald ( Zwi ) , Cinquanta ( Ita ) , Vazques Rana ( Mex ) , Boebka ( Oek ) , Sithole ( Zim ) .
